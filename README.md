# Задание 

Создать репозиторий для дисциплины на GitHub. Склонировать его себе на ПК. Написать свою первую программу. Ззапустить её. Сделать коммит и пуш.

# Описание проделанной работы
Я создала свой репозиторий на Github.С помощью консоли Git склонировала еготна свой ПК, для выполнения этой задачи я использовала `git clone`

После, я создала легкую программу, используя Visual Studio. Запустиоа ее и получила вывод программы. 

Позже, в консоли Git подготовила файл к коммиту, с помощью `git add`

После подготовки я сделала коммит и пуш, используя:
  `git commit`
  ;`git push`
