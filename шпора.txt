git --version -> версия гит 
git clone https://github.com/esinaaa/proga.git -> клонирование репозитория 
git clone <репозиторий> 
git add ./0.docx -> подготовка файла к коммиту 
git add . -> подготовка всех файлов сразу к коммиту
git status -> просмотр подготовки файлов (закоммичены ли они) 
git commit -m "0.docx" -> коммит файла 
git commit -m "<комментарий>" 
git push -> пуш коммита из локального репозитория (компик) в глобальный репозиторий (в github)
git pull -> "обратная операция push" загрузка коммита из глобального репозитория в локальный репозиторий 
git log -> просмотр изменений коммитов
